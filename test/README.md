# Image Finder

* 설명
  - 검색 키워드와 다양한 검색 옵션을 통해 조건에 맞는 이미지를 보여주는 서비스
  
* 사용 기술
  - getty image API
  - bulma v0.4.2
  - jQuery v2.1.1

* getty image API
  - [search API](http://developers.gettyimages.com/api/docs/v3/search/images/get/)

* 링크 : [Image Finder](https://dleogh29.github.io/Image_Finder)
